# 模拟RESTful风格案例

| 请求路径             | 请求方式   | 说明          |
|------------------|--------|-------------|
| `/employee`      | GET    | 获取所有员工的信息   |
| `/employee/{id}` | GET    | 根据ID获取员工的信息 |
| `/employee`      | POST   | 添加员工信息      |
| `/employee`      | PUT    | 更新员工信息      |
| `/employee/{id}` | DELETE | 根据ID删除员工信息  |
